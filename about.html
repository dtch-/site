DTCH(1)

SYNOPSIS

dtch [OPTION]

DESCRIPTION

This manual page documents the GNU version of dtch. dtch detaches quickly from other clients, often leaving many users confused as to its actual purpose.

OPTIONS

-i : ignore. -f : force. --f : forge.
